Utility 工具类

单一的基础元素或者结构

可(强制)实现特定的样式

如文字排版,特殊样式

工具命名

Utilities naming conventions

utilityName[-propertyName]

小写驼峰工具分类-小写驼峰工具属性

examples:

文字(txt)控制

<!-- 高亮突出 -->
<h1 class="txt-primary"></h1>

<!--右对齐  -->
<div class="txt-right"></div>

Component 组件类

一个组件包含

  • 根组件(块)
  • 子组件(子元素)
  • 组件样式(修或者主题)
  • 组件状态

一个页面由多个组件和组合或嵌套组成

组件命名

Components naming conventions

  • ComponentName
  • ComponentName-descendentName
  • ComponentName_modifierName
  • ComponentName.is-stateOfComponent
  • ComponentName-descendentName_modifierName

命名说明

  • class都以组件名开头
  • 组件名称大写驼峰，其他小写驼峰
  • 短线-连接组件与子元素(或者子模块) 最多一层
  • 下划线_连接修饰(特殊样式)
  • is-*标明特殊状态(可动态切换)

ComponentName-descendentName

例如导航(Nav)模块

<nav class="Nav">
    <a class="Nav-link"></a>
    <a class="Nav-link"></a>
</nav>

ComponentName_modifierName

如导航条(NavBar)

<!-- 普通导航条 -->
<div class="NavBar"></div>

<!-- 固定顶部导航条  -->
<div class="NavBar NavBar_top"></div>

ComponentName.is-stateOfComponent

如按钮(Btn)状态切换

<!-- 普通的按钮  -->
<botton class="Btn">正常</botton>

<!-- 禁用的按钮  -->
<botton class="Btn is-disabled">禁用</botton>
