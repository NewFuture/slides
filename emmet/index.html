层级关系

  • >子元素 (child)
  • +兄弟元素 (sibling)
  • ^返回上级 (climb-up)
  • ()组 (group)

> 子级

和CSS中一样,>表示子元素

nav>ul>li>a

<nav>
    <ul>
        <li><a href=""></a></li>
    </ul>
</nav>

+ 同级

和CSS中一样,>表示子元素

article>h1+section+section

<article>
    <h1></h1>
    <section></section>
    <section></section>
</article>

^ 父级

section>h2^section

<section>
    <h2></h2>
</section>
<section></section>

( )分组

(section>h2)+section

<section>
    <h2></h2>
</section>
<section></section>

复制

  • * 乘法
  • $ 自增量
  • @ 自增范围

*乘法

ul>li*3

<ul>
    <li></li>
    <li></li>
    <li></li>
</ul>

$ 自增

ul>li#list_$*3

<ul>
    <li id="list_1"></li>
    <li id="list_2"></li>
    <li id="list_3"></li>
</ul>

ul>li#list_$$$$*2

<ul>
    <li id="list_0001"></li>
    <li id="list_0002"></li>
</ul>

@ 起始

@-倒序,@n从n开始

ul>li#list_$@-*3

<ul>
    <li id="list_3"></li>
    <li id="list_2"></li>
    <li id="list_1"></li>
</ul>

ul>li#list_$@*2^ul>li#list_$@3*3

<ul>
    <li id="list_1"></li>
    <li id="list_2"></li>
</ul>
<ul>
    <li id="list_3"></li>
    <li id="list_4"></li>
    <li id="list_5"></li>
</ul>

Emmet + BEM规范

VScode 配置

{
    "emmet.preferences": {
        "bem.elementSeparator": "-",
        "bem.modifierSeparator": "_",
        "bem.shortElementPrefix": "-"
    },
    "emmet.syntaxProfiles": {
        "html": {
            "filters": "html, bem"
        }
    },
    "emmet.variables": {
        "lang": "zh"
    } 
}
