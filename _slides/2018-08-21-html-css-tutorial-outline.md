### How to Setup CSS

* Inline Styles
```html
<span style="color: red;">red text</span>
```
* Style Tag in HTML
```html
  <style>
    color {color: red;}
  </style>
```
* link CSS file
```html
<link href="style.css" type="text/css" rel="stylesheet">
```
