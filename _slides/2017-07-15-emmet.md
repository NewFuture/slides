## HTML缩写

~~

### HTML缩写

> `!` 然后按下<kbd>Tab</kbd>


自动生成默认的HTML模板

ps: `lang`和`charset`以及模板可自定义配置

```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <meta http-equiv="X-UA-Compatible" content="ie=edge">
    <title>Document</title>
</head>
<body>
    
</body>
</html>
```

~~

### 缩写规则

`tag:atrr` 冒号分割

如生成邮件链接
```
a:mail
```

```html
<a href="mailto:"></a>
```

## 层级关系

* `>`子元素 (child)
* `+`兄弟元素 (sibling)
* `^`返回上级 (climb-up)
* `()`组 (group)

~~
### `>` 子级

和CSS中一样,`>`表示子元素

```css
nav>ul>li>a
```
```html
<nav>
    <ul>
        <li><a href=""></a></li>
    </ul>
</nav>
```

~~
### `+` 同级
和CSS中一样,`>`表示子元素

```css
article>h1+section+section
```
```html
<article>
    <h1></h1>
    <section></section>
    <section></section>
</article>
```

~~
### `^` 父级
```css
section>h2^section
```

```html
<section>
    <h2></h2>
</section>
<section></section>
```

~~
### `(` `)`分组

```css
(section>h2)+section
```

```html
<section>
    <h2></h2>
</section>
<section></section>
```

## 复制

* `*` 乘法
* `$` 自增量
* `@` 自增范围

~~
### `*`乘法

```css
ul>li*3
```

```html
<ul>
    <li></li>
    <li></li>
    <li></li>
</ul>
```

~~
### `$` 自增

```css
ul>li#list_$*3
```

```html
<ul>
    <li id="list_1"></li>
    <li id="list_2"></li>
    <li id="list_3"></li>
</ul>
```

```css
ul>li#list_$$$$*2
```
```html
<ul>
    <li id="list_0001"></li>
    <li id="list_0002"></li>
</ul>
```

~~
### `@` 起始

`@-`倒序,`@n`从n开始
```css
ul>li#list_$@-*3
```
```html
<ul>
    <li id="list_3"></li>
    <li id="list_2"></li>
    <li id="list_1"></li>
</ul>
```
```css
ul>li#list_$@*2^ul>li#list_$@3*3
```
```html
<ul>
    <li id="list_1"></li>
    <li id="list_2"></li>
</ul>
<ul>
    <li id="list_3"></li>
    <li id="list_4"></li>
    <li id="list_5"></li>
</ul>
```

## Emmet + BEM规范

~~

### VScode 配置

```json
{
    "emmet.preferences": {
        "bem.elementSeparator": "-",
        "bem.modifierSeparator": "_",
        "bem.shortElementPrefix": "-"
    },
    "emmet.syntaxProfiles": {
        "html": {
            "filters": "html, bem"
        }
    },
    "emmet.variables": {
        "lang": "zh"
    } 
}
```

~~
