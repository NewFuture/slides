开源方案

  • Travis-CI (https://travis-ci.org)
  • Jenkins (https://jenkins.io)

以更新此slides为例

  1. 修改代码!(手动)
  2. 提交代码(commit & push)
  3. github 触发webhook 通知 Travis-CI
  4. Travis-CI 下载edit分支代码根据配置(.travis.yml)自动编译
  5. 编译成功将编译后的静态页面推送到master上
  6. 网站更新

  • fork 复制代码库

  • merge 合并代码(删除源分支)
